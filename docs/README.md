## 📋 Quick Reference

### 🔑 Key Features
- **50MB File Upload** - Support file hingga 50MB
- **Multi-tier Storage** - Cloudflare R2, Google Drive, Local
- **Smart Storage Manager** - Automatic tier selection
- **Unified Notifications** - Sistem notifikasi yang terintegrasi
- **Mobile-first Design** - Responsive dan touch-friendly

### 🛠️ Development
- **Framework**: Next.js 14 dengan App Router
- **Database**: Supabase (PostgreSQL)
- **Storage**: Multi-tier (Cloudflare R2, Google Drive, Local)
- **Real-time**: Socket.IO dan WebSocket
- **UI**: React 18 dengan Tailwind CSS
